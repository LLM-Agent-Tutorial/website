3.3 Overview

Automating biological data analysis through natural language

CellAgent is an autonomous, LLM-driven multi-agent framework that, given raw single-cell RNA sequencing (scRNA-seq) and spatial transcriptomics data plus natural language instructions, automates end-to-end analysis to deliver high-quality visualizations and processed outputs. Specialized agents plan, execute, and evaluate tasks, simulating a deep-thinking process to ensure transparent reasoning, iterative refinement, and clear results.

Role-play

Planner, Executor, Evaluator

Planner

A project manager with deep expertise in bioinformatics workflows. The planner interprets requests and outputs an ordered plan with rationale.

Executor

A bioinformatician who selects tools from the sc-Omni toolkit, generates Python/R code, and runs it inside a sandbox while explaining decisions.

Evaluator

A quality-control expert who inspects UMAP plots and metrics such as ARI/ASW, providing actionable feedback whenever outputs look suspect.

Planning & tool use

Expert-curated plans plus the sc-Omni toolkit

The Planner decomposes even simple requests into best-practice pipelines: quality control → normalization → highly variable genes → dimensionality reduction → clustering. The Executor then attaches the right tool.

Example · Planner output

Thought: To perform clustering, we must first ensure the data is properly prepared.
Plan: QC → Normalization → HVG → Dimensionality Reduction → Clustering.

Example · Tool selection rationale

"The current task is cell type annotation... I will select Cellmarker for this step as it provides a robust database of markers for annotation."

The agent then emits runnable code calling the documented functions from sc-Omni.

Reflection & memory

Evaluator-driven optimization and integrative memory

Example · Evaluator feedback

"The clusters in the UMAP visualization are overlapping, suggesting the current resolution parameter is too low to capture the underlying biological heterogeneity. The clustering is suboptimal. Re-run with a range of higher resolution values (e.g., 0.8, 1.0, 1.2) to achieve better separation."

CellAgent also employs global memory (the full analytical plan plus completed steps) and local memory (code, parameters, evaluator feedback for each sub-task). This ensures consistent cooperation and provides a retrievable trail for future iterations.
