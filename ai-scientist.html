Towards fully automated scientific research

The AI Scientist aims to independently carry out a scientific research process from idea generation to paper writing without any human involvement. It produces a research paper in three phases: idea generation, experimental iteration, and paper write-up. Afterward, an LLM-generated review assesses quality so the system can revise accordingly.

Planning

Planning helps the AI Scientist organize its multi-step workflow: experimental plans drive code execution while paper outlines guide write-ups. Below is the planning prompt straight from the paper.

Example · Planning experiments

Idea: Improving image classification through data augmentation.
Instruction: Plan the list of experiments you would like to run (max 5 runs).
Output: Prepare Data → Build Model → Train Baseline → Apply Augmentation → Train Again → Evaluate.

Tool use

In all three phases the AI Scientist leverages tools. During idea generation it retrieves literature via the Semantic Scholar API, during experiments it hands plans to the Aider coding assistant, and during paper write-up it compiles LaTeX into PDF. The excerpt below shows how queries are produced for retrieval.

Example · Generating a retrieval query

Instruction: "You have an idea and you want to check if it is novel. Provide a search query to survey the literature."

Output: Enhancing the image classification performance of convolutional neural networks using data augmentation.

The query becomes the parameter for a search_for_papers Python helper that returns Semantic Scholar metadata.

Reflection

Human scientific research relies heavily on reflection, and so does the AI Scientist. It rethinks ideas, analyzes experimental results, and performs two rounds of section-by-section reflection while writing. Crucially, the PDF manuscript is reviewed by an LLM reviewer that issues summary, strength, weakness, and decision fields:

Example · Automated review snippet

Summary: This paper explores the application of data augmentation techniques in image classification tasks...

Weaknesses: lacks comparison with state-of-the-art automatic data augmentation methods...

Overall: 4 · Decision: Reject.

The AI Scientist absorbs that feedback to improve subsequent drafts.
